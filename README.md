# libgit2cpp-conan

[Conan.io](https://www.conan.io/) package for [libgit2cpp library](https://github.com/AndreyG/libgit2cpp).

## About libgit2cpp

libgit2cpp is c++11 binding of [libgit2](https://github.com/libgit2/libgit2)
